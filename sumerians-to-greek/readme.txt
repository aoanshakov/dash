Имя моего домена

SUMERIANS-TO-GREEKS.RU

(это на всякий случай)

Хотелось бы, чтобы можно было запускать разные скрипты Пайтона для заполнения базы, а то я админку не делал, а стандартные админки не знаю, помогут ли. Я обычно делаю файл JSON и его использую для заполнения базы. Сервер базы данных MongoDB - это No SQL. Формат данных BSON. Это почти JSON. Документов в базе немного. Несколько тысяч, наверное, наберется всего, если все заполнить. Пока у меня в районе 2 тысяч маленьких документов. Поиск при наличии индексов очень быстрый.

Я использую dash. Это надстройка над flask. Локально dash работает очень хорошо. Никакого локального сервера не требует. Просто запускаешь скрипт, а потом входишь в Интернет под адресом localhost:8050. 

При деплойменте надо выдать приложение dash за приложение flask. Есть разные способы как это делать. В сети есть разные примеры.
Данные MongoDB расположены в папке db. Есть bat-файл, который позволяет поместить db в нужное место. Не знаю, правда, достаточно ли его просто скопировать и выполнить этот bat-файл?

В рабочей папке, кроме файла index.py должна содержаться подпапка assets в которой находится иконка сайта favicon.ico, файлы css и js. У меня также есть подпапка images. Но я ее не использую. Да, я использую еще некоторые пайтоновкие файлы как модули. Просто удобнее некоторые функции разместить в отдельном файле. index.py - это, конечно, говнокод, но у меня времени не было писать аккуратнее.
Какие модули Пайтона должны быть установлены, видно из преамбулы главного файла.
